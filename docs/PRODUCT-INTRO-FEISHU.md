# 认知飞轮 Cognitive Flywheel
### AI 外脑，和你一起成长

---

## 一句话定位

> **这不是一个工具，是一个和你一起成长的外脑。**
> 它不会替代你思考，而是记住你读过的所有东西、认识你的盲区、在你需要的时候用你自己的知识帮你做更好的决策。

---

## 为什么需要它？人脑的五个先天局限

我们都有同样的问题——

| 局限 | 表现 |
|------|------|
| 记忆衰减 | 看完一篇文章，三天后只记得 10% |
| 注意力有限 | 一次只能专注一件事，无法同时调用所有相关知识 |
| 知识是孤岛 | 投资、健康、创业——三个领域的知识互不相通 |
| 时间维度局限 | 我们只活几十年，而人类智慧积累了几千年 |
| 盲区不自知 | 不知道自己不知道什么，学习路径全靠运气 |

AI 时代的真正机会，不是让 AI 帮你做事，而是**用 AI 扩展你的认知边界**。

---

## 核心理念：认知飞轮

```
输入 → 记忆 → 思考 → 洞察 → 回流
  ↑                              │
  └──────────────────────────────┘
       每转一圈，外脑更懂你
```

**飞轮的关键**：每次思考产生的洞察，会自动压缩回流到记忆层。外脑越用越聪明，知道你关注什么、思考过什么、有哪些盲区——下次帮你思考时，会调用你自己积累的上下文，而不是泛泛而谈。

## 产品架构：三层设计

### 第一层：喂脑（Feed）

把你读过的文章、产生的想法"喂"给外脑：

- 粘贴链接（支持微信公众号、X/Twitter、YouTube、网页）
- 粘贴文章内容、输入想法
- 上传 PDF / Word 文件

外脑自动完成：
1. **提炼核心观点** — 3~5 个要点
2. **自动打标签** — 按领域、主题归档
3. **知识关联分析** — 找出与已有知识的关系（支持/矛盾/扩展/不同视角）
4. **跨域闪念** — 从其他领域给出类比灵感

> 示例：输入一篇关于"复利效应"的投资文章，外脑自动标记与已有知识的矛盾点，并给出"复利就像细胞分裂——每个子细胞立即成为新的分裂主体"的跨域类比。

### 第三层：思考室（Think）

四种外脑思考模式，全部基于你的记忆层驱动：

#### 🟦 圆桌会议 Roundtable
召集多位专家从不同视角挑战和完善你的想法。

- 每次召集 3 位真实历史人物或领域专家
- 基于你已有的知识注入上下文（"本次思考已引用 4 条记忆层上下文"）
- 专家的观点不是泛泛而谈，而是结合你存入的内容做个性化分析

> 示例：问"是否应该把积蓄投入股票市场"，系统召集巴菲特（价值投资）、彼得·林奇（普通投资者视角）、约翰·邓普顿（逆向投资），三人立场互相碰撞，给出截然不同的建议。

#### 🟧 认知教练 Cognitive Coach
发现你的盲区，生成个性化学习路径。

- 分析你现有知识的覆盖面和缺口
- 明确指出"你不知道自己不知道"的部分
- 给出 4 周可执行的学习计划（具体到书名/项目/练习）

> 示例：问"如何深入了解 AI Agent 的 Memory 模块"，教练告知已掌握架构概念，但缺失"向量嵌入的技术选型"和"记忆衰减机制"，并推荐 MemGPT 和 ChromaDB 实践路径。

#### 🟩 跨域连接 Cross-Domain
从其他领域借鉴灵感，发现意想不到的类比。

- 输入任意想法或问题
- 从生物学、历史、艺术、工程等领域中搜寻结构相同的现象
- 每个类比都有"操作映射"——如何把跨域洞察落地到你的问题

> 示例：研究"复利效应"，系统给出：细胞分裂（每个子细胞立即成为新的分裂主体）、音乐和声（乘法共振而非加法叠加）、建筑地基（看不见的底层决定顶层高度）三个类比。

#### 🟫 历史镜鉴 History Mirror
先驱们遇到过同样的问题，他们是怎么做的？

- 输入你正在面对的挑战
- 系统匹配人类历史上遭遇相似困境的人物
- 呈现他们的选择、结果和可迁移的洞察

> 示例：思考"如何从0建立个人品牌"，系统给出苏格拉底（被时代质疑但用真理建立了2000年的影响力）、宫本武藏（用一场关键战役打出声誉）、富兰克林（坚持每天凌晨4点写作，让时间成为盟友）。

### 飞轮自强化：洞察回流

每次思考结束，点击"存入记忆"：

- 飞轮 +1 转
- 本次思考的核心洞察压缩后存入记忆层
- 下次思考时，这些洞察会作为上下文被调用
- Me 页面实时更新飞轮转数、知识地图、盲区变化

**目前测试账号的飞轮已转 686 圈。**
