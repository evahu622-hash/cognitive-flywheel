# Cognitive Flywheel - 认知飞轮

> 这不是一个工具，是一个和你一起成长的外脑。

## 核心理念

人脑的局限：带宽有限、注意力有限、不擅长并行、记忆力有限。
时代的变化：算力不再稀缺、知识不再稀缺、技能重要性下降。

**Vision**: 用 AI 作为外脑帮助人脑减负，让人脑有限的带宽和算力用于更重要的事情和决策。

## 产品架构

```
    输入 → 记忆 → 思考 → 洞察
     ↑                    │
     └────── 回流 ────────┘
         每转一圈，外脑更懂你
         每转一圈，思考质量更高
         每转一圈，盲区更少
```

### 三层架构

#### 输入层
用户通过粘贴链接、文本、对话等方式将信息输入外脑。

#### 记忆层（地基）
- **知识库**：用户读过的、学过的一切
- **思考档案**：用户的观点、判断、推理过程
- **个人画像**：用户的领域、偏好、认知水平、盲区地图

#### 思考层（四大外脑能力）
1. **圆桌会议** — 多专家视角挑战你的想法（调用知识库+思考档案）
2. **认知教练** — 发现盲区，推荐个性化学习路径（调用个人画像+知识库）
3. **跨域连接** — 从其他领域借鉴灵感和方法（调用知识库跨领域检索）
4. **历史镜鉴** — 先驱们遇到过同样的问题吗？（调用思考档案匹配先驱经历）

### 自强化循环（认知飞轮）
思考层的所有输出都会压缩并回流到记忆层：
- 圆桌会议 → 记录你采纳了哪个观点、为什么、最终决策
- 认知教练 → 更新盲区地图（已补的不再推）
- 跨域连接 → 记录哪个跨域类比有启发
- 历史镜鉴 → 记录从哪个先驱经历中获得了什么

## 页面结构（V0）

| 页面 | 路由 | 功能 | 对应能力 |
|------|------|------|---------|
| Feed（喂脑） | /feed | 粘贴链接/文本，外脑自动消化入库 | 输入层 |
| Memory（记忆宫殿） | /memory | 知识库全局视图，关联、标签、成长轨迹 | 记忆层 |
| Think（思考室） | /think | 四大能力入口，选择模式后开始深度思考 | 思考层 |
| Me（认知画像） | /me | 知识地图、盲区地图、成长曲线 | 自强化可视化 |

## 技术栈

- **框架**: Next.js 15 (App Router)
- **UI**: shadcn/ui + Tailwind CSS
- **AI**: Vercel AI SDK（支持 Claude / GPT / Gemini / DeepSeek）
- **数据库**: Supabase (PostgreSQL + pgvector)
- **部署**: Vercel

## AI 模型策略

| 场景 | 推荐模型 | 原因 |
|------|---------|------|
| 深度推理（圆桌、教练） | Claude / GPT-4o / DeepSeek | 推理质量高 |
| 文章摘要、标签 | Haiku / GPT-4o-mini | 快且便宜 |
| 向量嵌入 | OpenAI Embedding | 知识检索用 |
